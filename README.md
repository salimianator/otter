# OTTER — Optimised Token-Text Extraction & Reduction

OTTER is a prompt compression research project that reduces long-context inputs
before they reach an LLM reader, preserving task-relevant information while
cutting token count.

## Project layout

```
otter/
├── data/
│   ├── inspect.py              # dataset stats & summary CSV
│   └── dataset_summary.csv     # generated by inspect.py
├── outputs/                    # results written here (git-ignored)
├── src/
│   ├── encoder.py              # SentenceEncoder (all-MiniLM-L6-v2)
│   ├── compress.py             # compress() stub — TBD
│   └── planner.py              # QueryPlanner stub — TBD
├── evaluate.py                 # QwenEvaluator (Qwen2.5-3B-Instruct)
└── requirements.txt
```

## Setup

```bash
python3 -m venv otter-env
source otter-env/bin/activate
pip install -r requirements.txt
python -m spacy download en_core_web_sm
```

## Download LongBench data

```python
from datasets import load_dataset

subsets = ['qasper', 'multifieldqa_en', 'qmsum', 'multi_news']
for subset in subsets:
    ds = load_dataset('THUDM/LongBench', subset, split='test', trust_remote_code=True)
    ds.to_json(f'data/longbench_{subset}.jsonl')
```

> **Note:** requires `datasets==2.21.0` — the newer 3.x/4.x series dropped
> support for the `LongBench.py` loading script.
